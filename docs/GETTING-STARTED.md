# Getting Started: HCI-101 Course Development
# はじめに：HCI-101コース開発

This guide walks through creating the HCI-101 course using the BMad BotEdu methodology.

このガイドでは、BMad BotEdu方法論を使用してHCI-101コースを作成する手順を説明します。

---

## Prerequisites | 前提条件

- BMad Method core installed in `.bmad-core/`
- bmad-botedu expansion pack installed
- Node.js and npm installed

---

## Course Development Steps | コース開発ステップ

### Step 1: BMad Installation ✅
Already complete - BMad framework is in `.bmad-core/`

### Step 2: Curriculum Generation
Generate comprehensive curriculum outline

### Step 3: Weekly Activities
Create detailed activity plans for each week

### Step 4: Final Project Guidelines
Develop project requirements and rubric

### Step 5: Bot Configuration
Configure HCI-101 Bot with RAG and workflows

### Step 6: Lecture Materials
Create lecture guides for instructors

Follow the BMad BotEdu agents to complete each step interactively.
