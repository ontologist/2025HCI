1. Usability Test Plan

  • Test Objectives: Clear, specific goals
  • Participant Criteria: Well-defined target users
  • Test Tasks: 5 realistic, unbiased tasks
  • Success Criteria: Measurable metrics for each task
  • Metrics Planning: What you'll measure (task success, time, errors, SUS)
  • Test Script: Introduction, instructions, and closing

2. Practice Test Observation Notes

  • Detailed observations from practice test
  • User quotes and behavioral notes
  • Evidence of think-aloud protocol
  • Issues discovered during practice
  • Refinements made to test plan

3. Bot Interaction Log

Screenshots showing bot guidance during test planning
